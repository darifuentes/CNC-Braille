🔧 Máquina de Escritura Braille

Sistema automatizado para impresión de texto Braille

PIC 18F550 Motores a Pasos Nextion HMI Sistema Braille

🎯 Objetivo del Proyecto

Desarrollar una máquina automatizada capaz de convertir texto digital en escritura Braille sobre papel opalina, proporcionando una solución tecnológica accesible para personas con discapacidad visual.

⚡ Innovación Tecnológica

Integración de microcontroladores, actuadores mecánicos de precisión y interface humano-máquina intuitiva para crear un sistema de escritura Braille completamente automatizado.

🔧 Especificaciones Técnicas

Microcontrolador: PIC 18F550 - Control principal del sistema
Actuadores: Motores a pasos con husillos para movimiento lineal
Interface: Nextion HMI - Control táctil intuitivo
Sustrato: Papel opalina - Superficie de impresión Braille

🚀 Funcionamiento del Sistema

El usuario ingresa el texto a través de la interface Nextion HMI. El microcontrolador PIC 18F550 procesa la información, convierte cada carácter a su equivalente en Braille y coordina el movimiento de los motores a pasos. Los husillos transforman el movimiento rotativo en desplazamiento lineal, posicionando con precisión el mecanismo de punzado sobre el papel opalina para crear los puntos en relieve del sistema Braille.
